# 📚 Library Management System (C# Console App)

A simple console-based Library Management System written in C#.  
This project was created as a learning pet-project to practice OOP, collections, file handling, and clean architecture.

---

## 🚀 Features

- Add new books
- Show all books
- Add readers
- Show all readers
- Borrow books
- Return books
- Save and load data using JSON
- Input validation and error handling
- Data persistence between app restarts

---

## 🛠 Technologies Used

- C#
- .NET Console Application
- Object-Oriented Programming (OOP)
- Collections (`List<T>`)
- File I/O
- JSON Serialization (`System.Text.Json`)
- Exception handling (`try/catch`)

## 💾 Data Persistence

The application automatically saves data to JSON files:
- `books.json`
- `readers.json`
- `loans.json`

Data is loaded on application startup and saved before exit.

## ▶️ How to Run

1. Clone the repository
2. Open the solution in Visual Studio
3. Run the project
4. Use the console menu to interact with the system
